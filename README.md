# React To-Do app

> "React To-Do app" is a simple To-Do list created using React library.


![screenshot](./public/Screenshot-app.png.png)

## Built With

- React


## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Node.js
- npm

### Setup

Clone the repository with:

```
git clone git@github.com:ShinobiWarior/react-todo-app.git
```
or download [ZIP file](https://github.com/ShinobiWarior/react-todo-app/archive/refs/heads/main.zip)

### Install
To install all dependencies run:
```
npm install
```
### Usage
To run it on the browser you can run in the project directory:

 ```
 npm start
 ```

## Run tests 
To test the project please run in the project directory:

```
npm test
```
